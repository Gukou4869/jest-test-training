🚀 Jest 練習用の Next.js プロジェクト　 🚀

Jest（React 　 Testing Library React）を用いた Jest を書く訓練用 Reposiory です。

## 背景

現職で当初は Unit テストを書いていたのだが、納期などに追われ結局テストをしだいに書けなくなり。。。QA エンジニアの方に QA を任せているような状況でありました。
が、プロジェクトが一通り落ち着き、反省点として最低限 Unit テストを書いた方が良かったと実感しているので、まずは Jest の使い方に際勉強したいと思い、この Repository を立ち上げました 🔥🔥🔥

## 目標

Jest での一通りの基本的な書き方をマスターすることを目標とします。

## Test 方法

```bash
npm run jest
# or
yarn jest
# or
```
